# Menu Images

1. This folder stores all menu item images
2. Each image filename must match the slug in the JSON
3. Preferred format is .jpg
4. Example:
   - alaska-roll.jpg
   - amaebi.jpg
   - hibachi-teriyaki-filet-mignon.jpg
5. If an image is missing, the frontend should use a fallback placeholder
6. Do not use spaces or uppercase letters in filenames
